## <h2 id="runLocally" >Run Locally </h2>

Clone the project

```bash
  git clone https://github/Nik4Furi/03_allReactHooks
```

Go to the project directory

```bash
  cd 03_allReactHooks
  cd Backend
```
Install dependencies

```bash
  yarn
```

Start the server

```bash

  yarn dev (Run or restart, whenever you save any file(js))
```
